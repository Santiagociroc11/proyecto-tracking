Política de Privacidad de HotAPI

Última actualización: 1 de Agosto de 2024

Bienvenido a HotAPI ("nosotros", "nuestro"). Nos comprometemos a proteger tu información personal y tu derecho a la privacidad. Si tienes alguna pregunta o inquietud sobre nuestra política o nuestras prácticas con respecto a tu información personal, contáctanos.

Esta política de privacidad se aplica a toda la información recopilada a través de nuestros servicios (que incluyen nuestra aplicación web), así como a cualquier servicio, venta, marketing o evento relacionado.

1. ¿QUÉ INFORMACIÓN RECOPILAMOS?

Recopilamos información personal que nos proporcionas voluntariamente cuando te registras en la aplicación, expresas interés en obtener información sobre nosotros o nuestros productos y servicios, o cuando te pones en contacto con nosotros.

La información personal que recopilamos incluye:

  • Información de contacto: Nombre, dirección de correo electrónico.
  • Credenciales: Contraseñas y datos de seguridad similares utilizados para la autenticación y el acceso a la cuenta.
  • Datos de productos y seguimiento: IDs de Píxel de Facebook, tokens de acceso a APIs (cifrados), y configuraciones de seguimiento que nos proporcionas para el funcionamiento de nuestros servicios.

Integración con la API de Meta (Facebook)

Si decides conectar tu cuenta de Meta a nuestro servicio, recopilaremos la siguiente información para proporcionar la funcionalidad de seguimiento de gastos y cálculo de ROAS:

  • Token de Acceso: Recibimos un token de acceso de Meta para realizar llamadas a la API en tu nombre. Este token se almacena de forma segura utilizando cifrado AES-256-GCM.
  • ID de Cuenta Publicitaria y de Negocio: Almacenamos el ID de tu cuenta publicitaria de Meta y el ID de tu negocio para asociar los datos correctamente con tu producto.
  • Datos de Campañas y Rendimiento: Utilizando los permisos `ads_read` y `read_insights`, accedemos periódicamente a los datos de tus campañas publicitarias. Específicamente, solicitamos el gasto (`spend`) de tus campañas para calcular métricas de rendimiento como el ROAS. No almacenamos detalles específicos de las creatividades o audiencias de tus anuncios.

El uso de estos datos se limita estrictamente a mostrarte tu gasto publicitario y el ROAS calculado dentro de tu dashboard personal en nuestra aplicación.

2. ¿CÓMO UTILIZAMOS TU INFORMACIÓN?

Utilizamos la información personal recopilada para diversos fines comerciales, que se describen a continuación:

  • Para proporcionar y gestionar nuestros servicios: Incluyendo la configuración del seguimiento de eventos y la sincronización con las APIs de terceros como Hotmart y Meta.
  • Para gestionar tu cuenta: Para crear y gestionar tu cuenta de usuario y mantenerla en funcionamiento.
  • Para calcular y mostrar métricas de rendimiento: Específicamente, para obtener el gasto de tus campañas de Meta y calcular el ROAS, que es una característica central de nuestro servicio.
  • Para comunicarnos contigo: Para enviarte información administrativa, como cambios en nuestros términos, condiciones y políticas.

3. ¿SE COMPARTIRÁ TU INFORMACIÓN CON ALGUIEN?

Solo compartimos información con tu consentimiento, para cumplir con las leyes, para proporcionarte servicios, para proteger tus derechos o para cumplir con obligaciones comerciales.

Nunca venderemos tu información personal a terceros.

4. ¿CÓMO MANTENEMOS TU INFORMACIÓN SEGURA?

Hemos implementado medidas de seguridad técnicas y organizativas apropiadas diseñadas para proteger la seguridad de cualquier información personal que procesamos. Damos especial importancia a la seguridad de los tokens de acceso de API, los cuales se cifran en reposo en nuestra base de datos.

Sin embargo, a pesar de nuestras salvaguardias y esfuerzos para asegurar tu información, ninguna transmisión electrónica por Internet o tecnología de almacenamiento de información puede garantizarse como 100% segura.

5. ¿CUÁLES SON TUS DERECHOS DE PRIVACIDAD?

Tienes derecho a revisar, cambiar o cancelar tu cuenta en cualquier momento. Si deseas solicitar la revisión o eliminación de tu información personal, puedes contactarnos utilizando la información de contacto proporcionada a continuación.

6. CAMBIOS A ESTA POLÍTICA DE PRIVACIDAD

Podemos actualizar esta política de privacidad de vez en cuando. La versión actualizada se indicará con una fecha de "Última actualización" y la versión actualizada será efectiva tan pronto como sea accesible.
